# Navigator CFO — Brand Style

## Visual Theme & Atmosphere
Premium financial advisory brand. Deep navy paired with warm cream creates a trustworthy, sophisticated atmosphere. Minimalist, corporate-editorial design with generous whitespace, professional photography (business settings, handshakes), and a clean compass-themed logo. The feel is authoritative yet approachable — a modern CFO firm that instills confidence.

## Colors
| Role | Hex | Notes |
|---|---|---|
| Background (dark) | `#0A2340` | Deep navy — primary brand color, used for hero sections and major content blocks |
| Background (light) | `#F5EDE1` | Warm cream — header, alternate sections, light-mode backgrounds |
| Primary | `#0A2340` | Navy — dominant brand color |
| Accent (one only) | `#7D7F69` | Olive/sage — used for secondary buttons and subtle accents |
| Text Primary (on light) | `#151515` | Near-black for body text on cream/white |
| Text Primary (on dark) | `#FFFFFF` | White for text on navy backgrounds |
| Text Muted | `#6B6B6B` | Mid-gray for captions and secondary text |
| Border | `#D4CFC7` | Warm neutral border |

**Color scheme:** dual — dark navy sections alternating with warm cream sections
**Accent rule:** one accent (olive #7D7F69) for emphasis. Never two.

## Typography

- **Headings:** Montserrat — Google Fonts. Clean geometric sans-serif. Fallback: Arial, sans-serif
